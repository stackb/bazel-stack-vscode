## Syntax Highlighting

<p></p>

Highlighting of `bazel` files is activated for the following file types:

- `BUILD`
- `BUILD.bazel`
- `*.BUILD`
- `WORKSPACE`
- `*.bzl`
- `*.star`
- `*.sky`

Highlighting of `bazelrc` files is activated for:

- `*.bazelrc`
